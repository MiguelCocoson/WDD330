Chapter 5: Objects

  • Object: self-contained set of related values and functions.
  • Creates an empty object: new Object();
  • Objects work similarly to dictionaries in python:
  • "object.key" gives the value.
  • "object[value]" also returns the value.
  • "undifined" is the error obtain when trying to acces a property that does not exist.
